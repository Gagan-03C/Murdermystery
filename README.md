# Murdermystery

Interactive, frontend-only investigation game for college events.

## Structure
- `index.html`: landing page with teaser + rules popup.
- `dashboard.html`: investigation dashboard with dynamic evidence + terminal.
- `data.js`: narrative and evidence content.
- `clues.js`: game state and clue progression.
- `terminal.js`: command parser and clue validation flow.
- `vault.js`: final vault with attempt limiter and ending.
-  `style.css`: shared styling for landing and dashboard.
